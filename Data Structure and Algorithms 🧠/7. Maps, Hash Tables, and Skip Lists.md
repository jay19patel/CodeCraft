# Maps, Hash Tables, and Skip Lists

## 1. Maps and Dictionaries
A **map** (also called a **dictionary** in Python) is a collection of key-value pairs. Each key must be unique, and it maps to a specific value.

---

## 2. The Map ADT (Abstract Data Type)
**Operations:**
- `M[k]`: Retrieve value associated with key `k`
- `M[k] = v`: Set the value `v` for key `k`
- `del M[k]`: Remove the key `k` and its value
- `len(M)`: Get the number of items
- `k in M`: Check if `k` is in map

---

## 3. Application: Counting Word Frequencies
Useful application of a map is counting how many times each word appears in a document using a dictionary.

---

## 4. Pythonâ€™s MutableMapping Abstract Base Class
Provides a base for custom map-like classes. It requires implementation of:
- `__getitem__`
- `__setitem__`
- `__delitem__`
- `__iter__`
- `__len__`

---

## 5. MapBase Class
```python
class MapBase(MutableMapping):
    class _Item:
        __slots__ = '_key', '_value'

        def __init__(self, k, v):
            self._key = k
            self._value = v

        def __eq__(self, other):
            return self._key == other._key

        def __ne__(self, other):
            return not (self == other)

        def __lt__(self, other):
            return self._key < other._key
```

## 6. Unsorted Table Map (Simple Implementation)
```python
class UnsortedTableMap(MapBase):
    def __init__(self):
        self._table = []

    def __getitem__(self, k):
        for item in self._table:
            if k == item._key:
                return item._value
        raise KeyError(k)

    def __setitem__(self, k, v):
        for item in self._table:
            if k == item._key:
                item._value = v
                return
        self._table.append(self._Item(k, v))

    def __delitem__(self, k):
        for j in range(len(self._table)):
            if k == self._table[j]._key:
                self._table.pop(j)
                return
        raise KeyError(k)

    def __len__(self):
        return len(self._table)

    def __iter__(self):
        for item in self._table:
            yield item._key
```

## 7. Hash Table Concepts
### Hash Function:
- A function that maps keys to indices of the internal array

### MAD Compression:
- `(hash(k) * scale + shift) % prime % N`

### Collision Resolution:
- **Separate Chaining** (Linked List in each bucket)
- **Open Addressing** (e.g., Linear Probing)

---

## 8. HashMapBase (Abstract)
```python
class HashMapBase(MapBase):
    def __init__(self, cap=11, p=109345121):
        self._table = cap * [None]
        self._n = 0
        self._prime = p
        self._scale = 1 + randrange(p - 1)
        self._shift = randrange(p)

    def _hash_function(self, k):
        return (hash(k) * self._scale + self._shift) % self._prime % len(self._table)

    def __len__(self):
        return self._n

    def __getitem__(self, k):
        j = self._hash_function(k)
        return self._bucket_getitem(j, k)

    def __setitem__(self, k, v):
        j = self._hash_function(k)
        self._bucket_setitem(j, k, v)
        if self._n > len(self._table) // 2:
            self._resize(2 * len(self._table) - 1)

    def __delitem__(self, k):
        j = self._hash_function(k)
        self._bucket_delitem(j, k)
        self._n -= 1

    def _resize(self, c):
        old = list(self.items())
        self._table = c * [None]
        self._n = 0
        for (k, v) in old:
            self[k] = v
```

## 9. Chain Hash Map
```python
class ChainHashMap(HashMapBase):
    def _bucket_getitem(self, j, k):
        bucket = self._table[j]
        if bucket is None:
            raise KeyError(k)
        return bucket[k]

    def _bucket_setitem(self, j, k, v):
        if self._table[j] is None:
            self._table[j] = UnsortedTableMap()
        old_size = len(self._table[j])
        self._table[j][k] = v
        if len(self._table[j]) > old_size:
            self._n += 1

    def _bucket_delitem(self, j, k):
        bucket = self._table[j]
        if bucket is None:
            raise KeyError(k)
        del bucket[k]

    def __iter__(self):
        for bucket in self._table:
            if bucket is not None:
                for key in bucket:
                    yield key
```

## 10. SortedTableMap
```python
class SortedTableMap(MapBase):
    def __init__(self):
        self._table = []

    def _find_index(self, k, low, high):
        if high < low:
            return high + 1
        else:
            mid = (low + high) // 2
            if k == self._table[mid]._key:
                return mid
            elif k < self._table[mid]._key:
                return self._find_index(k, low, mid - 1)
            else:
                return self._find_index(k, mid + 1, high)

    def __getitem__(self, k):
        j = self._find_index(k, 0, len(self._table) - 1)
        if j == len(self._table) or self._table[j]._key != k:
            raise KeyError(k)
        return self._table[j]._value

    def __setitem__(self, k, v):
        j = self._find_index(k, 0, len(self._table) - 1)
        if j < len(self._table) and self._table[j]._key == k:
            self._table[j]._value = v
        else:
            self._table.insert(j, self._Item(k, v))

    def __delitem__(self, k):
        j = self._find_index(k, 0, len(self._table) - 1)
        if j == len(self._table) or self._table[j]._key != k:
            raise KeyError(k)
        self._table.pop(j)

    def __iter__(self):
        for item in self._table:
            yield item._key

    def reversed(self):
        for item in reversed(self._table):
            yield item._key
```

## 11. Skip Lists - Search and Update Operations
A **skip list** allows fast search within an ordered sequence of elements using a layered linked list.

### Search Operation
- Start from the top-level forward pointer.
- Traverse forward while the next node is less than the search key.
- Drop down a level when traversal cannot proceed.
- Continue until the bottom level is reached.

### Insert Operation
- Perform a search to determine the position.
- Generate a random level for the new node.
- Insert at each level up to the generated level.

### Delete Operation
- Search for the element.
- Remove the node at all levels.
